PROJECT TITLE: SetDestroyer!
PURPOSE OF PROJECT: Solve the infamous set game found at
  https://www.setgame.com/set/puzzle
VERSION or DATE: Fall 2016
HOW TO START THIS PROJECT: 
  javac -cp . *.java
  java -cp . Set
AUTHORS: Jeromey Klein
USER INSTRUCTIONS: Enter in the features for each set card and the 
  program will solve the current configuration of the set game.
